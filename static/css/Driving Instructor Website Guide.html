What is React?

React (or React.js) is a JavaScript library used to build user interfaces especially for modern, interactive web applications. It was developed by Facebook (now Meta).

What Does React Do?

React helps you:

  • Build dynamic, responsive, and interactive UIs

  • Break down your UI into reusable components (like Lego blocks)

  • Update only the parts of the page that change this makes apps faster and more efficient

How React Fits Into a Web App

Here's a simple view of how a web app might be structured:

PartExample Technology
Frontend (UI)React.js
Backend (Logic)Django (Python)
DatabasePostgreSQL
API (data bridge)Django REST Framework (DRF)

So, with React, your frontend talks to your backend API to get or send data. This is called a Single Page Application (SPA).

Example Use Case (for your driving instructor site):

If you used React on the frontend:

  • The student sees a calendar and clicks on a date

What is TensorFlow?

TensorFlow is an open-source machine learning framework developed by Google. It's designed for building and training machine learning models especially deep learning models like neural networks.

What Can TensorFlow Do?

TensorFlow can help you:

  • Build image recognition systems

  • Create natural language processing (NLP) models (like chatbots, text summarizers)

  • Develop time-series forecasting models (like predicting stock prices or traffic)

  • Build recommendation systems

  • Train custom AI models for your own datasets

Whats Inside TensorFlow?

  • TensorFlow Core: Low-level API for building machine learning models from scratch.

  • Keras (built-in): High-level API to build and train deep learning models easily. Most developers use this.

  • TensorBoard: A tool for visualizing training performance, metrics, and more.

  • tf.data: Tools for loading and processing datasets efficiently.

Real Example (How Its Used)

Lets say you want to build a model to classify different hair types in images (like you mentioned before curly, straight, no hair, etc.):

With TensorFlow:

  1. Load your dataset (images of hair types).

  2. Preprocess the data (resize, normalize).

  3. Build a CNN (Convolutional Neural Network) model using Keras.

  4. Train the model on your data.

  5. Evaluate its performance.

  6. Use it to predict new images.

from django.db import models
from accounts.models import CustomUser

class Booking(models.Model):
    student = models.ForeignKey(CustomUser, on_delete=models.CASCADE, related_name='student_bookings')
    instructor = models.ForeignKey(CustomUser, on_delete=models.CASCADE, related_name='instructor_bookings')
    date = models.DateField()
    time = models.TimeField()
    status = models.CharField(max_length=20, choices=[
        ('booked', 'Booked'),
        ('completed', 'Completed'),
        ('cancelled', 'Cancelled'),
    ], default='booked')
    created_at = models.DateTimeField(auto_now_add=True)

    def __str__(self):
        return f"{self.student.username} -> {self.instructor.username} on {self.date} at {self.time}"

Step 3: Create Student Dashboard View

In dashboard/views.py, update student_dashboard:

python
from django.shortcuts import render
from bookings.models import Booking
from django.utils import timezone

def student_dashboard(request):
    user = request.user
    today = timezone.now().date()

    upcoming = Booking.objects.filter(student=user, date__gte=today, status='booked').order_by('date')
    past = Booking.objects.filter(student=user, date__lt=today).order_by('-date')

    return render(request, 'dashboard/student_dashboard.html', {
        'upcoming': upcoming,
        'past': past,
    })

Step 2: Add the Cancel View

In dashboard/views.py:

python
from django.shortcuts import get_object_or_404
from django.http import HttpResponseRedirect
from django.urls import reverse
from bookings.models import Booking

def cancel_booking(request, booking_id):
    booking = get_object_or_404(Booking, id=booking_id, student=request.user)
    booking.status = 'cancelled'
    booking.save()
    return HttpResponseRedirect(reverse('student_dashboard'))
